# Team Photos

Drag and drop headshots into this folder using these exact filenames:

| Role                              | Filename       |
|-----------------------------------|----------------|
| Founder & President               | `founder.jpg`  |
| VP of Student Affairs             | `vp.jpg`       |

**Accepted formats:** `.jpg`, `.jpeg`, `.png`, `.webp`

If you use a different extension (e.g. `.png`), open `about.html`, find the `<img>` tag for that person, and update the file extension to match.

Photos are displayed as circular crops at ~120px (founder) and ~100px (VP), so square images work best. The page automatically falls back to a lettered placeholder if a file isn't found.
